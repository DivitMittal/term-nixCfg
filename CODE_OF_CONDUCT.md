# Code of Conduct

Be respectful, constructive, and welcoming. This project has a zero-tolerance policy for harassment, discrimination, or hostile behavior of any kind.

## Expected Behavior

- Use welcoming and inclusive language
- Accept constructive feedback gracefully
- Focus on what is best for the project and its users

## Unacceptable Behavior

- Personal attacks, insults, or derogatory comments
- Harassment in any form, public or private
- Publishing others' private information without consent

## Enforcement

Violations may be reported by opening a private GitHub issue or contacting the maintainer directly. Maintainers reserve the right to remove comments, close issues, or ban contributors who violate this policy.
