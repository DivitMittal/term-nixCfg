# emulators/kitty

[Kitty](https://sw.kovidgoyal.net/kitty/) terminal emulator configuration.

| File | Role |
|---|---|
| `kitty.conf` | Main config (vim-folded sections) |
| `current-theme.conf` | Active color theme — managed by `kitty +kitten themes`, not edited directly |
| `launch-actions.conf` | Protocol handlers for file/image URLs |

These files are used directly from this directory (symlinked or copied by the OS-nixCfg home-manager config).
